# Controlador Tradução

O controlador de tradução trabalha recebendo do menu o código da linguagem que selecionamos no menu e trocando a linguagem do projeto. Linguagens essas que são adicionadas na configuração de locale do projeto na godot.

## Função:

A única função existente, que foi criada pela gente, é a set_traducao. Que pega a linguagem selecionada no menu e seta a tradução do jogo para ela.
