# Documentação Componentes

A ideia deste documento é apresentar a documentação dos componentes do nosso jogo, com foco em explicar o que eles fazem e como utilizá-los. Isso nos ajudará a realizar melhores implementações no futuro.

Componentes são scripts adicionados a objetos (ou entidades) do mundo do jogo para que executem comportamentos específicos.
